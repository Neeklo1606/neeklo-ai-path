База знаний Neeklo: инструкция для пользователя

Эта страница помогает настроить базу знаний из Obsidian и подключить ее к ассистенту.

Шаг 1. Настройка модели

  1. Откройте Админка → База знаний.
  2. Выберите ассистента.
  3. Провайдер: OpenAI API (рекомендуется).
  4. Вставьте ключ OpenAI.
  5. Нажмите Загрузить доступные модели и выберите модель.
  6. Нажмите Сохранить шаг 1.
Зачем Base URL? Обычно не нужен. Он нужен только если вы используете не официальный OpenAI, а OpenAI-compatible сервис.

Шаг 2. Загрузка знаний

  1. Для Obsidian используйте Импорт Obsidian vault (папка с .md).
  2. Либо загрузите одиночные файлы .pdf .txt .docx .md.
  3. Можно вставлять текст вручную в поле «Добавить текст».

Шаг 3. Проверка

  • Смотрите блок «Точек в базе» — число должно расти после загрузки.
  • Граф знаний отображает связь фрагментов и обновляется автоматически.
  • Если чат не учитывает базу — проверьте, что у ассистента статус «Активен» и модель сохранена.

LLM-помощник

На странице есть блок LLM-помощник по настройке. Туда можно писать вопрос простым языком, например:

  • «Почему у меня нет ответа в чате?»
  • «Как правильно загрузить Obsidian?»
  • «Какую модель выбрать для продаж?»
